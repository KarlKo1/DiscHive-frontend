# DiscHive

A demo e-commerce site.

**Backend code [link](https://github.com/KarlKo1/DiscHive-backend)**.

## Features

- Sign Up/Login
- View items
- Add items to cart
- Go through a demo buying process
- View orders

## Technologies Used

- React
- Nextjs
- MongoDB
- Styled Components
- Framer Motion
- Strapi
- Postgres
- GraphQL
- Auth0
- Stripe
- Cloudinary
